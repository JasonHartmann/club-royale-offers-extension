Install & Use

  1. Install from the Chrome Web Store (link coming soon).
  2. Log into your Royal Caribbean account and open the Club Royale Offers page.
  3. Click the added Show Casino Offers button in the top bar.
  4. Sort columns or switch to grouped accordion mode.
  5. Highlight expiring offers and scan newest instantly.
  6. Close with the X or press ESC.

No data leaves the page. Processing is local and ephemeral.

Privacy & Scope

The extension only reads the existing HTML and API response already delivered to the page while you are authenticated. It does not transmit credentials, tokens, or analytics. You may disable it anytime; it is idle when not on the Offers page.
