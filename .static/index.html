Install & Use

  1. Install from the Chrome Web Store.
  2. Log into your Royal Caribbean or Celebrity account and open the Club Royale or Blue Chip Club Offers page.
  3. Click the added Show All Offers button in the top bar.
  4. Switch between profiles using the tabs at the top.
  5. Link two profiles with the chain icon to see combined offers.
  6. Sort columns, group, or filter as needed.
  7. Hide groups with the trash icon (per profile).
  8. Close with the X or press ESC.

No data leaves the page. Processing is local and ephemeral.

Privacy & Scope

The extension only reads the existing HTML and API response already delivered to the page while you are authenticated on Royal Caribbean or Celebrity. It does not transmit credentials, tokens, or analytics. You may disable it anytime; it is idle when not on the Offers page.
