How do I add a second profile and link them?

  1. Open the offers page while signed into the first loyalty account and click Show All Offers.
  2. Select the Profiles tab in the header bar, then press “Add Profile”.
  3. Give the profile a short label (e.g., “Player A”), choose a highlight color, and hit Save. The extension stores the minimal ID locally.
  4. Log out of the cruise site completely, sign into the second loyalty account, navigate to the offers page, and click Show All Offers again. This log-out/log-in sequence is the only way to capture another profile.
  5. Once both profiles exist, click the link chain icon on both the header tabs.
  6. The new Combined tab becomes enabled. It surfaces offers that can be upgraded or combined between those two accounts.

Tip: To refresh a profile’s data later, you must redo the same sign-out/sign-in + Show All Offers flow for that account, then hit Refresh Profile in the Profiles tab. If you rename a profile, re-open the linking dialog to keep the combined tab label accurate.

How does the Back-to-Back (B2B) tool work and what do depths mean?

The B2B Builder reads the sailings that are already in your table, and labels each row with a depth number. That number is simply the longest uninterrupted chain of cruises you can sail starting with that offer using what you currently have in your account. In other words, a depth of 4 means you could board that sailing and keep hopping from ship to ship four sailings in a row before you run out of eligible comps.

Depth is influenced by the Side-by-Side toggle in the builder. When Side-by-Side is on, the planner assumes you are comfortable switching ships on the same day in the same port, so the chain length can continue even if two sailings overlap on embarkation day. When it is off, the builder only considers sailings available for the same ship, so the depth number may shrink accordingly.

Clicking the depth pill for an offer will display the Back-to-Back Builder tool. Use the tool to construct a chain of sailings for destinations you care about, then click Add to Favorites. All sailings in the chain will get added to the Favorite tab with a unique code for that chain in the B2B column

How do I search by upgrade price?

  1. Open the Advanced Search drawer by clicking the button.
  2. Select one of the custom search criteria from "Interior Price", "Ocean View Price", "Balcony Price", or "Suite Price" in the "Add Field" drawer.
  3. Set the less than operator on the new filter.
  4. Type the max price you are willing to pay for the upgrade (e.g., 250 for $250).
  5. The filter applies instantly to the selected profile and respects whatever group or sort state you already have.

Why is the table slow and how can I speed it up?

  • Use the Hidden Groups feature to hide large sections of the table. To do this, open the group picker via the Yellow Folder icon in each column header and use the "Trashcan" icon to hide those groups. For the best results, hide groups with many rows that you know you can't use, such as "Code: 25TIER3".
  • Open Settings and toggle Auto-run Back-to-Back Builder Calculations to limit depth calculations for the B2B tool.

If performance stays rough, send your browser/OS combo to admin@comproyale.com so we can look into it.
